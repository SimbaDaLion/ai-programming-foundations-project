# AI Programming Foundations Project

## Project Description
This project demonstrates a complete data workflow including data cleaning, exploratory analysis, and visualization using the Iris dataset. The workflow serves as a foundation for future machine learning and AI projects.

## Dataset
**Iris Flower Dataset** - 150 rows, 5 columns
- Source: https://archive.ics.uci.edu/ml/datasets/iris

## How to Run

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run the Notebook
1. Launch Jupyter Notebook:
```bash
jupyter notebook
```
2. Open data_workflow.ipynb
3. Run all cells (Kernel → Restart & Run All)

## Future ML/AI Integration
This workflow can be extended for machine learning by:
- Adding feature engineering
- Splitting data into train/test sets
- Implementing classification models (Logistic Regression, SVM, etc.)
